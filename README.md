# Molly Olly's Wishes

Molly Olly's Wishes is a charity who help children with life threatening illnesses.

More information on the charity can be found [on their website](https://mollyolly.co.uk).

This repo contains code for some key systems built for Molly Olly's.

## Data Management System

A Node.JS app which stores the wishes and therapeutic product requests the charity receives.

[Data Management System](https://github.com/molly-ollys/data-management-system)

### Searchdown

A JS library for searchable dropdowns built specifically for the DMS. Packaged and published to npm.

[Searchdown](https://github.com/molly-ollys/searchdown)

## Silent Auction App

An app which allows users to bid on auction lots. Being rebuilt in 2026 using Svelte + C#.

[Current Project](https://github.com/molly-ollys/silent-auction-app) (2026 rebuild)

[Original Project](https://dev.azure.com/mollyollys/Molly%20Ollys%20Auction) (ask Joe for access if needed)

## 500 Club

A new website which allows users to sign up, automatically pay monthly or manually pay month by month, tracks payments and generates monthly winners for admins.

[500 Club](https://github.com/molly-ollys/500-club)
